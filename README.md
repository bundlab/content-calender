Modern full-stack **Content Calendar & Publishing Management System**
Built with **FastAPI** (backend), **React + TypeScript + Vite** (frontend), **SQLModel**, **PostgreSQL**, and **Docker**.

Clean architecture • Type-safe • Great DX • Production-ready foundation.

## ✨ Features

- 🐍 FastAPI + SQLModel – type-safe models & auto-generated OpenAPI
- ⚛️ React 18 + TypeScript + Vite – fast development & build
- 🔐 JWT Authentication + role-based permissions
- 📅 Content planning, scheduling & publishing workflow
- 📦 One-command local dev with Docker Compose
- 🧪 pytest backend + React Testing Library frontend
- 📊 TanStack Query – powerful data fetching & caching
- 🛠 GitHub Actions CI – lint, test, build
- 📝 Clean, modular folder structure

## 🚀 Quick Start (Local)

```bash
# 1. Clone
git clone https://github.com/YOUR_USERNAME/content-calender-cms.git
cd content-calender-cms

# 2. Copy env file
cp .env.example .env

# 3. (optional) Adjust .env values
#    POSTGRES_PASSWORD, JWT_SECRET_KEY, BACKEND_CORS_ORIGINS, etc.

# 4. Start stack
docker compose up -d --build

# 5. Open in browser:
#    • Frontend     →  http://localhost:5173
#    • API docs     →  http://localhost:8000/docs
#    • Alternative docs →  http://localhost:8000/redoc
```
